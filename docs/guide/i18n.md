# 国际化

内置语言：`zh-CN`、`en-US`。

## 基本用法

```vue
<YanivEditor locale="zh-CN" />
<YanivEditor locale="en-US" />
<YanivInlineEditor locale="en-US" v-model:content="html" />
```

`normalizeLocaleCode` 映射：

- `zh-TW` / `zh-HK` → `zh-CN`
- `en` → `en-US`
- 其它 `zh*` → `zh-CN`，`en*` → `en-US`

## 多实例隔离

同页多个编辑器可设不同 `locale`，互不影响。Demo：`/multi-instance`。

## 自定义文案

```ts
import { createI18n, loadLocale, ensureLocalesLoaded } from "@yanivjs/yaniv-editor";

const i18n = createI18n({
  locale: "zh-CN",
  messages: {
    "zh-CN": { editor: { placeholder: "自定义占位…" } },
  },
});
```

扩展层通过 session 内 `ctx.locale` 静态快照读取文案；Chrome 组件通过 `inject` 读取实例 locale。**禁止**在组件内直接调用全局 `t()`。

## locale 与 session

`locale` 变化会改变 `sessionKey`，触发 session 重建。
